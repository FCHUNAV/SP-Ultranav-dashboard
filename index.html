⬇️ Exportar & Publicar en GitHub Pages
① Contraseña y exportar

Los datos se encriptan con AES-256. Sin la contraseña, el archivo es ilegible — ni en el código fuente.

② Publicar en GitHub Pages (una sola vez)
  1. Crear cuenta gratis en github.com
  2. New repository → nombre ultranav-dashboard → Public
  3. Settings → Pages → Source: main branch → Save
  4. Upload el HTML descargado → renombrar a index.html
  5. URL permanente: [usuario].github.io/ultranav-dashboard
③ Para actualizar los datos
  1. Carga los nuevos Excel aquí en Claude → genera
  2. Click ⬇️ Re-exportar → descarga el HTML nuevo
  3. En GitHub → sube el archivo → reemplaza el index.html
  4. La misma URL muestra los datos nuevos al instante ✓
